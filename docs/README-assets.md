# Diagram assets (`docs/images/`)

These **original** SVG figures support the main [README](../README.md). They are **not** copies of SAFe copyrighted artwork; they summarize concepts in a neutral style for documentation and presentations.

| File | Intent |
|------|--------|
| [`images/pipeline.svg`](images/pipeline.svg) | Four aspects of the Continuous Delivery Pipeline as used in this app’s taxonomy: Continuous Exploration → Continuous Integration → Continuous Deployment → Release on Demand. |
| [`images/architecture.svg`](images/architecture.svg) | Runtime view: browser SPA, FastAPI, persistence, OpenAI review, export outputs. |
| [`images/radar.svg`](images/radar.svg) | Illustrative spider/radar chart: map `results.json` practice scores (or domain averages) to axes in BI tools. |
| [`images/heatmap.svg`](images/heatmap.svg) | Illustrative grid: pipeline area × practice with score bands; incomplete cells can be styled separately using `practice_completion_status`. |

To replace a diagram, keep the same filename or update the image references in `README.md`.
